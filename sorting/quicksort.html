Quick sort

sort follows Divide and Conquer algorithm. It is dividing elements in to smaller parts based on some condition and performing the sort operations on those divided smaller parts. Hence, it works well for large datasets. So, here are the steps how Quick sort works in simple words. 1. First select an element which is to be called as pivot element. 2. Next, compare all array elements with the selected pivot element and arrange them in such a way that, elements less than the pivot element are to its left and greater than pivot is to it's right. 3. Finally, perform the same operations on left and right side elements to the pivot element.
